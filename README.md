# NCAIC CLI Toolkit

A collection of tools for NCAIC competitors.

## Installation

```sh
npm i -g ncaic
```

## Usage

Checkout the help menu:

```sh
ncaic --help
```

Verify the `team.json`:

```sh
ncaic verify
```

Test the agent program:

```sh
ncaic test
```

Run the local server:

```sh
ncaic run
```

Check the language support of current environment:

```sh
ncaic check
```
